
# Web Hosting with GCP

This Terraform infrastructure is deployed in Google Cloud Platform.

Services Used:
- Cloud Storage
- Cloud CDN
- Cloud Load Balancing
- Cloud DNS

## Deployment

To deploy this project run

```terraform
  terraform init
  terraform plan
  terraform apply
```
